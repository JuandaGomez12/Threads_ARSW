# Threads_ARSW

## Objetivo
- En esta actividad se requiere que la tematica vista en clase que es la de los hilos se evalue cuanto tiempo se demora imprimir hasta el numero 5000000, la idea es ir incrementando el numero y evaluar sus tiempos y obtener conclusiones adecuadas.
